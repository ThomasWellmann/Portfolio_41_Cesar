Thomas Wellmann da Riva Cesar's Sortier Algotithmen

Start mit wählen zwischen eine zufällige Liste oder eine, der man selber eingibt.

4 Sortier Augorithmen:
1: Sequential Sort
2: Bubble Sort
3: Insertion Sort
4: Heap Sort

Weitere Funktionen:
1: Add new number (keep sorting style)
2: Increscent
3: Decrescent
4: Zig-Zag
5: Shuffle (unsort)

An bestimmte stellen: mit "ESC" startet man das Programm neu.
